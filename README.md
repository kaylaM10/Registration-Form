# Registration Form

## DESCRIPTION
-This project is a simple front-end registration form built using HTML and CSS. It was created as a practice exercise.

## TECHNOLOGIES USED
-html
- css

## FEATURES 
-User input fields for first name, last name, email, and password
- Account Type
- File to upload a profile picture
- Age input with min and max limits
- Dropdown menu for referral source
- Bio textarea
- Terms and conditions checkbox
- Submit button

## HOW TO RUN THE PROJECT
1. Clone the repo:  git clone https://github.com/kaylaM10/Registration-Form.git
2. Open the project folder in VS Code 
3. Install Live Server by Ritwick Dey in Extension
4. Right click on index.html and select live server to view in your browser
